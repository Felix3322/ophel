# ⚡ 体验增强

除了三大核心功能，Ophel 还提供了 20 多项实用的体验增强功能，全面提升 AI 对话效率。

## Claude 专属增强

### Session Key 管理

::: warning Claude 用户专属
此功能仅适用于 Claude (claude.ai) 平台。
:::

一键管理 Claude Session Key，支持多账号轮询：

- 🔑 **快速切换**：在不同 Session Key 之间一键切换
- 🔄 **自动轮询**：多个 Key 轮流使用，避免单一账号限制
- 🔐 **安全存储**：Key 加密存储，支持 WebDAV 同步备份
- 📊 **状态监控**：实时显示各 Key 的可用状态

## 界面定制

### 宽屏模式

自定义页面最大宽度，适配超宽屏显示器：

| 设置   | 说明           | 示例               |
| ------ | -------------- | ------------------ |
| 百分比 | 相对于屏幕宽度 | `80%`、`90%`       |
| 像素值 | 固定宽度       | `1200px`、`1600px` |
| 无限宽 | 填满屏幕       | `100%`             |

### 气泡宽度

独立设置用户提问气泡的宽度，让对话更易阅读：

- 📏 自定义宽度：`60%` ~ `100%`
- 🎯 对齐方式：左对齐 / 居中 / 右对齐

### 侧边栏布局

针对 AI Studio 的特殊优化：

- 📂 自动折叠导航栏
- ⚙️ 自动折叠设置面板
- 📐 最大化工作区域

## 阅读辅助

### 滚动锁定

防止 AI 生成时页面意外滚动：

- 🔒 生成时锁定视口
- 👆 手动滚动时暂停锁定
- ⚙️ 快捷键 `Alt + S` 手动切换

### 阅读历史

自动记录并恢复上次阅读位置：

- 📍 每次离开自动保存位置
- 🔄 返回对话时自动恢复
- 📊 支持多个对话独立记录

### Markdown 优化

修复和增强 Markdown 渲染：

- ✅ 修复 Gemini 的加粗/代码块渲染问题
- ✅ 实时渲染用户输入框中的 Markdown
- ✅ 优化代码块语法高亮

## 内容交互

### 公式复制

双击 LaTeX 公式直接复制源码：

```latex
E = mc^2
```

- 🖱️ 双击公式 → 自动复制
- 📋 复制 LaTeX 源码到剪贴板
- ✨ 显示复制成功提示

### 表格转换

一键将 AI 回复中的表格转换为 Markdown 格式：

| 表格      | 转换结果      |
| --------- | ------------- |
| HTML 表格 | Markdown 表格 |
| 格式完整  | 可直接使用    |

### 水印移除

自动移除 Gemini/AI Studio 生成图片中的隐形水印：

::: warning 权限要求
此功能需要授予 `<all_urls>` 权限，可在权限管理中配置。
:::

- 🖼️ 自动处理生成的图片
- 🔇 移除隐形数字水印
- 📥 下载干净的图片
