# 核心功能

Ophel 提供三大核心功能，帮助你更高效地使用 AI 对话平台。

## 🧠 智能大纲

自动解析对话内容，生成结构化导航树，让长对话阅读不再迷失。

[详细了解 智能大纲 →](./outline)

**核心能力：**

- 多层级导航 — 自动解析用户问题与 H1-H6 标题
- 智能过滤 — 只显示用户问题，快速回顾对话脉络
- 滚动跟随 — 自动高亮当前阅读位置
- 一键跳转 — 点击大纲节点瞬间定位

---

## 💬 会话管理

为海量对话提供强大的组织和管理能力。

[详细了解 会话管理 →](./conversation)

**核心能力：**

- 无限滚动 — 加载所有历史会话
- 标签系统 — 为会话添加自定义标签
- 彩色文件夹 — 用色彩区分不同类型会话
- 多格式导出 — Markdown / JSON / TXT
- WebDAV 同步 — 跨设备数据同步

---

## ⌨️ 提示词助手

让常用提示词触手可及，支持变量模板实现动态内容。

[详细了解 提示词助手 →](./prompt)

**核心能力：**

- 变量模板 — 定义 `{{变量}}`，使用时自动弹窗填写
- 分类管理 — 自定义分类并自动分配色彩
- 快捷访问 — 置顶常用、查看最近使用
- Markdown 预览 — 实时预览渲染效果
