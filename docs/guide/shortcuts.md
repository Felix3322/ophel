# ⌨️ 快捷键

Ophel 提供丰富的快捷键支持，帮助你高效操作 AI 对话界面。所有快捷键均可在设置中自定义。

## 导航

| 快捷键    | 功能       |
| --------- | ---------- |
| `Alt + T` | 滚动到顶部 |
| `Alt + B` | 滚动到底部 |
| `Alt + Z` | 返回锚点   |

## 面板

| 快捷键    | 功能              |
| --------- | ----------------- |
| `Alt + P` | 切换面板显示      |
| `Alt + D` | 切换明暗主题      |
| `Alt + 1` | 切换到第 1 个标签 |
| `Alt + 2` | 切换到第 2 个标签 |
| `Alt + 3` | 切换到第 3 个标签 |

## 大纲

| 快捷键            | 功能          |
| ----------------- | ------------- |
| `Alt + R`         | 刷新大纲      |
| `Alt + E`         | 展开/折叠全部 |
| `Alt + Shift + 1` | 展开到 1 级   |
| `Alt + Shift + 2` | 展开到 2 级   |
| `Alt + Shift + 3` | 展开到 3 级   |
| `Alt + Q`         | 切换用户视角  |
| `Alt + ↑`         | 上一个标题    |
| `Alt + ↓`         | 下一个标题    |
| `Alt + L`         | 定位大纲      |
| `Alt + F`         | 搜索大纲      |

## 会话

| 快捷键             | 功能         |
| ------------------ | ------------ |
| `Ctrl + Shift + O` | 新建会话     |
| `Alt + Shift + R`  | 刷新会话列表 |
| `Alt + Shift + L`  | 定位当前会话 |
| `Alt + [`          | 上一个会话   |
| `Alt + ]`          | 下一个会话   |

## 编辑

| 快捷键             | 功能             |
| ------------------ | ---------------- |
| `Ctrl + Shift + E` | 导出对话         |
| `Ctrl + Shift + C` | 复制最新回复     |
| `Alt + ;`          | 复制最后代码块   |
| `Alt + S`          | 锁定滚动         |
| `Alt + I`          | 聚焦输入框       |
| `Alt + K`          | 停止生成         |
| `Alt + /`          | 打开模型选择菜单 |

## 设置

| 快捷键           | 功能                    |
| ---------------- | ----------------------- |
| `Alt + \`        | 快捷键一览              |
| `Alt + ,`        | 打开设置                |
| `Ctrl + Alt + C` | 打开 Claude 专属设置    |
| `Ctrl + Alt + S` | 切换 Claude Session Key |
| `Ctrl + Alt + G` | 打开 Gemini 专属设置    |
| `Ctrl + Alt + T` | 打开外观主题设置        |
| `Ctrl + Alt + L` | 打开模型锁定设置        |

## 自定义快捷键

你可以在设置面板中自定义所有快捷键：

1. 按 `Alt + \` 或打开设置面板
2. 选择「快捷键」标签
3. 点击任意快捷键进行编辑
4. 按下新的按键组合
5. 点击保存

::: tip 提示
如果快捷键与网站原生快捷键冲突，可以修改为其他组合，或者暂时禁用快捷键功能。
:::
